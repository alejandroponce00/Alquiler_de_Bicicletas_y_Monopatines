# Sistema de Alquiler de Bicicletas y Monopatines

Proyecto desarrollado en C++ utilizando Programación Orientada a Objetos para gestionar el alquiler de bicicletas y monopatines eléctricos.

## Funcionalidades

- Gestión de clientes
- Gestión de vehículos
- Gestión de empleados
- Registro de alquileres y reservas
- Devolución y cancelación de alquileres
- Informes y estadísticas
- Validaciones de datos y disponibilidad

## Tipos de Vehículos

- Bicicletas
- Monopatines eléctricos

## Características del Sistema

- Menú principal con submenús
- Almacenamiento en archivos binarios `.dat`
- Altas, bajas lógicas, modificaciones y consultas
- Control de disponibilidad de vehículos
- Cálculo automático del monto por bloques de 15 minutos
- Validaciones de clientes, empleados y fechas

## Informes Disponibles

- Recaudación mensual
- Vehículo más alquilado
- Cliente con más alquileres
- Cantidad de alquileres por tipo de vehículo

## Tecnologías Utilizadas

- Lenguaje: C++
- Paradigma: Programación Orientada a Objetos
- Persistencia: Archivos binarios `.dat`

## Objetivo

Desarrollar un sistema de consola capaz de administrar de forma eficiente el alquiler de vehículos eléctricos mediante una estructura modular y organizada.
